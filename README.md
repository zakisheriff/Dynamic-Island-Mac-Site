---

## 🌟 Vision

Dynamic Island Mac's mission is to be:

- **A completely free productivity tool** — no subscriptions, no data tracking, 100% open
- **A seamless system integration** — feels like it was built by Apple for macOS
- **A beautiful, modern interface** — leveraging advanced animations and glassmorphism

---

## ✨ Why Dynamic Island Mac?

The MacBook notch has been "dead space" for years.  
Dynamic Island Mac democratizes the premium iOS experience by bringing **fluid music controls, smart notifications, and system alerts to every Mac user for free**.

---

## 🎨 Apple-Inspired "Liquid Glass" Design

- **Minimalist Aesthetics**  
  Pure CSS and Framer Motion implementation following Apple's design principles — focused and elegant.

- **Liquid Glass Effects**  
  Advanced `backdrop-filter` and translucent layers create a native, high-end feel.

- **Soft Elevation & Haptics**  
  Subtle shadows and micro-interactions that make the interface feel tactile and alive.

- **System Typography**  
  Native `-apple-system` and SF Pro-style typography for a perfect system match.

---

## 🚀 Key Features

- **Music & Media Control**  
  Interactive "Now Playing" expanded view with album art and playback controls.

- **Smart Notifications**  
  System-level notifications transformed into elegant, non-intrusive island expansions.

- **Timers & Clock**  
  Live countdowns and alerts integrated directly into the notch.

- **Face ID Transitions**  
  Premium onboarding flow featuring Face ID-inspired success animations.

- **High Frame-Rate Performance**  
  Optimized with Framer Motion for buttery-smooth 120Hz interaction on ProMotion displays.

## 🚀 Quick Start

### Prerequisites

- **Node.js** (v18+)
- **NPM** or **PNPM**

### 1. Clone the Repository

```bash
git clone https://github.com/zakisheriff/Dynamic-Island-Mac-Site.git
cd Dynamic-Island-Mac-Site
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Application

```bash
npm run dev
```

Visit **http://localhost:3000** 🎉
